# Jubayer Ahmed — Personal Portfolio

A clean, Apple-inspired personal portfolio website built with pure HTML, CSS, and vanilla JavaScript. No frameworks, no dependencies — just fast, elegant, and fully responsive.

## Live Preview

> Deploy to GitHub Pages → `https://MainReturn0.github.io`

---

## Features

- Frosted-glass navigation bar (Apple-style)
- Smooth scroll-reveal animations on all sections
- Fully responsive (mobile, tablet, desktop)
- Hero section with animated entrance and gradient background
- About section with stat cards
- Skills categorized by domain
- Project cards with stack tags
- Education timeline with institution flags and badges
- Language proficiency bars
- Contact section with email, GitHub, and phone links

## Getting Started

### Run locally

Just open the file in your browser:

```bash
open index.html
```

Or use VS Code's Live Server extension.

### Deploy to GitHub Pages

1. Create a repository named `MainReturn0.github.io`
2. Push `index.html` to the `main` branch
3. Go to **Settings → Pages → Source → main branch**
4. Your site will be live at `https://MainReturn0.github.io` ✅

---

## Customization

All content is in a single `index.html` file. To update:

- **Colors** → edit CSS variables in `:root` at the top of `<style>`
- **Content** → find the relevant `<section>` and edit the text
- **Projects** → duplicate a `.project-card` block and update details
- **Links** → search for `href=` and update emails, GitHub, LinkedIn URLs
